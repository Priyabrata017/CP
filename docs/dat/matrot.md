
## Matrix rotations : 
#### Rotate by +90:
1. Transpose
2. Reverse each row

### Rotate by -90:
**Method 1 :**
1. Transpose
2. Reverse each column

**Method 2 :**
4. Reverse each row
5. Transpose

### Rotate by +180:
**Method 1:**
* Rotate by +90 twice

**Method 2:**
* Reverse each row and then reverse each column (Transpose)

### Rotate by -180:
**Method 1:**
* Rotate by -90 twice

**Method 2:**
* Reverse each column and then reverse each row

**Method 3:**
* Rotate by +180 as they are same
